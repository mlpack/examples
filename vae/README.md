```cpp
arma::mat dataset;
data::Load("dataset.csv", dataset, true);
```

Now, we will take the features in a seperate matrix. In Armadillo representation, the rows and columns will be interchanged.

```cpp
arma::mat data = dataset.submat(1, 0, train.n_rows - 1, train.n_cols - 1);
```
